🚀 StreamMonitor

抖音直播自动监控 & 微博通知系统

📋 程序概览
StreamMonitor 是一个全自动化的抖音直播监控系统，能够：
  • 持续监控抖音直播间状态
  • 直播开启时自动发布微博通知
  • 直播结束时发送详细的统计数据
  • 记录所有通知历史日志
  • 提供 REST API 接口查询状态
  • 24/7 不间断运行监控

🔄 工作原理
三步监控流程：
  • 检测状态： 每30秒检查一次直播间状态
  • 识别变化： 检测离线→在线 或 在线→离线
  • 发送通知： 自动发布微博并记录详细日志
核心功能：
  • 使用 Douyin API 实时查询直播状态
  • 通过微博 Cookie 自动发布通知
  • 完整的统计数据（观看数、点赞、粉丝等）
